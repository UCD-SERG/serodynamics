# Excluding population parameters

`ex_popparams` excludes estimated population parameters from final
output [data.frame](https://rdrr.io/r/base/data.frame.html).

## Usage

``` r
ex_popparams(x)
```

## Arguments

- x:

  A [data.frame](https://rdrr.io/r/base/data.frame.html) with a
  `Subject` variable.

## Value

A filtered [data.frame](https://rdrr.io/r/base/data.frame.html)
excluding population parameters.
