# Adds attributes

`add_jags_attrs` adds specified attributes to a
[data.frame](https://rdrr.io/r/base/data.frame.html).

## Usage

``` r
add_jags_attrs(df, attrs)
```

## Arguments

- df:

  A [data.frame](https://rdrr.io/r/base/data.frame.html) of the original
  input dataset.

- attrs:

  [attributes](https://rdrr.io/r/base/attributes.html) to attach to the
  [data.frame](https://rdrr.io/r/base/data.frame.html).

## Value

A [data.frame](https://rdrr.io/r/base/data.frame.html) with specified
[attributes](https://rdrr.io/r/base/attributes.html) attached.
